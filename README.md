# nba-data-analysis
Data analysis project using SQL and Tableau to evaluate NBA teams from 2000-2023 to analyze win percentage and win consistency. 
# NBA Team Performance Analysis

## Overview
This project analyzes NBA team performance from 2000–2024 using SQL and Tableau. The goal was to evaluate overall team success and consistency across seasons.

## Tools Used
- SQL (data cleaning and aggregation)
- Tableau (data visualization)

## Dashboard
![Dashboard](dashboard.png)

## Key Analysis
- Ranked teams by average win percentage
- Compared best and worst seasons to evaluate consistency
- Identified differences in long-term performance across teams

## Insights
- Teams like the Spurs and Celtics show strong overall performance over time
- Some teams demonstrate high variability between their best and worst seasons
- Performance consistency varies significantly across franchises

## Project Summary
This project demonstrates the ability to clean data, perform analysis using SQL, and build a clear, insight-driven dashboard using Tableau.
